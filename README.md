# bitrix-code-analysis

Скрипт для оценки объёма исходного кода (число файлов, строк, размер в байтах) в редакциях «Старт» и «Бизнес» на основе открытого ядра https://github.com/avshatalov48/bitrix-core-business.

Для редакции «Старт» было сопоставлено 13 папок из modules, то есть метрика не совсем точная.
